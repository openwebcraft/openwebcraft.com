Title: Raspberry Pi OS internet access via iOS USB tethering

----

Text:

1. Add `dtoverlay=dwc2` to the `/boot/config.txt`
2. Add `modules-load=dwc2,g_ether` to the end of `/boot/cmdline.txt` (right behind `â€¦rootwait`)
3. Check devices: `networkctl` 
4. Create file `/etc/systemd/network/ipad.link` :

```sh
[Match]
Name=usb0

[Network]
DHCP=yes
```

5. Restart systemd networking: `sudo systemctl restart systemd-networkd`
6. Check configuration `networkctl`
