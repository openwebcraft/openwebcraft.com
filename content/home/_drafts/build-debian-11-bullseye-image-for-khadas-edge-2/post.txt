Thanks to the quite excellent documentation provided by *Khadas*, the process is pretty simple and straight forward.

One simply starts from the GitHub repo [khadas/fenix](https://github.com/khadas/fenix), which provides a one-stop script to build Ubuntu/Debian images for *Khadas* ARM SBC:

```bash
# install debs
sudo apt-get install git make lsb-release qemu-user-static

# create project dir and clone fenix script
mkdir -p ~/Code/khadas-edge2
cd ~/Code/khadas-edge2
git clone --depth 1 https://github.com/khadas/fenix
cd fenix

# adjust environment variables in file config-template.conf
KHADAS_BOARD=Edge2
LINUX=5.10
UBOOT=2017.09
DISTRIBUTION=Debian
DISTRIB_RELEASE=bullseye
DISTRIB_TYPE=gnome
DISTRIB_ARCH=arm64
INSTALL_TYPE=EMMC

# load environment configuration from file
source env/setenv.sh config config-template.conf

# build image
make
```

Put the `edge2-debian--gnome-linux-5.10-fenix-1.5-230501-emmc-develop.img.xz` file onto a USB stick, start  the Edge2 into*oowow* (on start hold `FUN`, press and release `RST` key), and write the image onto EMMC.
